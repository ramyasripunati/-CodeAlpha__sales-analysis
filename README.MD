Data Science & Machine Learning Project
1. Project Objective
The goal of this project is to predict future sales based on key factors including advertising spend, target segment,
and platform. By applying machine learning regression techniques, we analyze how marketing investments translate
into revenue, enabling businesses to optimize their advertising strategies.
